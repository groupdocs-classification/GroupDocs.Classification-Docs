# GroupDocs.Classification - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Classification](https://docs.groupdocs.com/classification/) static documentation.

## About GroupDocs.Classification

[GroupDocs.Classification](https://products.groupdocs.com/classification/) is a text and document classification SDK that enables you to categorize raw text and documents into predefined categories using IAB-2 and document taxonomies.

Available for .NET.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/classification/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
